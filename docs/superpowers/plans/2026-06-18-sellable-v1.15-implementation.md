# Sellable v1.15 — Trust/Polish + Cost Widget — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans (inline, recommended for this project — live browser/backend verification) or superpowers:subagent-driven-development. Steps use `- [ ]` checkboxes.

**Goal:** Turn DeepotusVideoGen v1.15 into a credible paid one-time download for crypto creators: close trust/polish gaps and add a Cost Widget (per-API usage + remaining + monetary value, and a preview budget before any spend).

**Architecture:** FastAPI backend (Python 3.14, `%LOCALAPPDATA%\DeepotusVideoGen\backend\app`), compiled React bundle (`frontend/dist/assets/index-BEOJX8L5.js`, minified — patched via unique-anchor Python scripts), SQLite job store, BYO keys in `%LOCALAPPDATA%\DeepotusVideoGenData\.env`. Re-bundled into the Inno Setup installer + portable export.

**Tech stack:** FastAPI, httpx, Pillow, aiosqlite/SQLAlchemy; React (minified); Inno Setup; Node 24 (`node --check`), embedded Python 3.14 runtime.

**Project conventions (ADAPTED — read first):**
- **No git** in this environment → instead of `git commit`, snapshot each edited file to `<file>.bak` before changing, and keep the bundle's `.pre-<task>.bak`.
- **No pytest** → backend "tests" are standalone scripts run with the embedded runtime: `& "$env:LOCALAPPDATA\DeepotusVideoGen\runtime\python\python.exe" test_x.py` using bare `assert` + prints. Endpoint tests = `Invoke-RestMethod` against the running backend.
- **Frontend** has no source → each UI task = (1) grep the bundle to find the exact unique anchor, (2) Python unique-anchor replace (assert count==1, write .bak), (3) `node --check` the bundle copied to `.mjs`, (4) hard-reload the app and verify via `javascript_tool` DOM/network that the control fires.
- **Restart backend** after any backend edit: stop PID on :8765, clear `__pycache__`, relaunch `python -m uvicorn app.main:app --host 127.0.0.1 --port 8765` hidden.
- **Verify-before-done**: never mark a task done without the Verify step passing.
